
# AI Quiz App

An interactive, AI-inspired quiz application built with HTML, CSS, and JavaScript. Features adaptive difficulty, smooth animations, and a glass-like UI for an engaging user experience.

## Features

- **Adaptive Questions**: Difficulty adjusts based on user performance (simulating AI).
- **Smooth Animations**: Includes confetti for correct answers, shakes for incorrect ones, and glass-like effects.
- **Real-Time Scoring**: Displays score updates after each question.
- **Responsive Design**: Works on desktop and mobile devices.
- **10 Questions**: Covers general knowledge topics.
